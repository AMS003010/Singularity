### Run the project ( &nbsp; _just wanna run it_ &nbsp; )
#### Prerequisites
- rustc
- Cargo
- rustup

You can install them [here](https://www.rust-lang.org/tools/install)

Run this command to build the executable and run it
```
cargo r -r
```
